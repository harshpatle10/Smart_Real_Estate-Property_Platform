# E-Nagar – Smart Real Estate & Building Approval Platform 🏢

E-Nagar is a microservices-based smart property and building approval platform developed using Java, Spring Boot, React.js, Tailwind CSS, MySQL, and JWT Authentication. The system digitizes property management and building approval workflows for citizens and government authorities.

## 🚀 Features

* Citizen registration and property application management
* Building approval workflow for Engineers and Officers
* Admin dashboard for monitoring applications and users
* Secure JWT-based authentication & role-based access control
* Document upload and verification system
* Application tracking and status updates
* Responsive modern UI dashboards
* API Gateway integration for centralized routing

## 🛠️ Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Java, Spring Boot, Spring Security
* Architecture: Microservices, API Gateway
* Database: MySQL
* Authentication: JWT
* Tools: Git, GitHub, Maven

## 🔧 Microservices Included

* Auth Service
* Application Service
* Review Service
* Admin Service
* API Gateway
